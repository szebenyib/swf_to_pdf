# swf_to_pdf
Converts swf to pdf
Also converting svg to pdf

# Installation
`pip3 -r requirements.txt`

# Usage
Copy the .py file to the folder with svg files, then:

* simply run it
`python3 swf_to_pdf.py`

* ask for help
`python3 swf_to_pdf.py --help``

* add some parameters
`python3 swf_to_pdf.py --x_size 1536 --y_size 2048`
